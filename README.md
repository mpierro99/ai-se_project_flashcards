# Flashcard App

My first project in TripleTen's AI-Assisted Software
Engineering program. It includes decks of flashcards,
each of which can be viewed in a carousel.

## Features

- A gallery of study decks, each with its own color and card count
- A carousel view for flipping through a deck's cards (question and answer)
- Hash-based routing between the home, carousel, and 404 views
- The ability to delete a deck from the gallery
- Decks and cards rendered dynamically from data using an HTML template

## Technologies used

- HTML
- CSS (with BEM naming convention)
- Vanilla JavaScript (ES modules)
- The HTML `<template>` element for dynamic rendering
- Git and GitHub (deployed with GitHub Pages)
